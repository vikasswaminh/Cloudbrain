# Cloudbrain AI Gateway & Execution Platform

A comprehensive, Cloudflare-native platform for AI task execution, dynamic model management, and CLI-based automation.

## Project Structure
- **/coding-brain-backend**: Cloudflare Workers, Hono, D1, Durable Objects.
- **/coding-brain-frontend**: React (Vite) dashboard for Admin and Users.
- **/coding-brain-cli**: Node.js/Executable CLI for terminal-based AI pair programming.
