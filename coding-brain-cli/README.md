# Coding Brain CLI

The Sovereign FSM-based CLI for the Coding Brain platform.

## Installation
See [INSTALL.md](./INSTALL.md) for detailed instructions.

## Usage
```bash
brain --help
```

## Features
- **Natural Language Commands**: "Create a React app"
- **YOLO Mode**: `brain -y "do it"`
- **Expert Modes**: Security, Architect, DevOps
- **Local Sovereignty**: execution happens locally, state is synced.
